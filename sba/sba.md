## Requirements

> Create your application locally, initialize a local git repo, make frequent commits. When complete, push to GitHub and submit the link.

| Requirement | Weight |
|-------------|--------|
| Use the **fetch API or Axios** to communicate with an external web API. Use the data to populate your application's content and features. | **20%** |
